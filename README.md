**JOGO DE XADREZ**

Um jogo de xadrez desenvolvido em C#, como projeto pessoal para praticar lógica, orientação a objetos e estrutura de classes, com o objetivo de simular um tabuleiro interativo e movimentação correta das peças.

**Funcionalidades**

- Tabuleiro 8x8 com visual em console/CLI (ou GUI se for o caso)  
- Movimentação das peças conforme regras do xadrez (peões, torres, bispos, cavalo, rainha, rei)  
- Captura de peças  
- Verificação de xeque / xeque-mate / empate
- Alternância de turno entre dois jogadores  

**Tecnologias utilizadas**

- C# (.NET)  
- Orientação a Objetos  
- Estruturas de dados básicas (listas, matrizes)  

**Como executar / Rodar o projeto**

1. Clone o repositório:  
   ```bash
   git clone https://github.com/matheusdomepass/Jogo-de-xadrez.git
   ```  
2. Abra a solução/projeto no Visual Studio (ou editor de sua preferência).  
3. Compile/build o projeto.  
4. Execute o programa (via “Start” ou executável).  
5. Siga as instruções no console / interface — selecione peça, destino, etc.
